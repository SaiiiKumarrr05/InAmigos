## Sections Included
1. ✅ Navbar — Logo, links, gradient CTA, mobile hamburger (CSS only)
2. ✅ Hero — Headline, description, 2 CTAs, animated AI illustration, gradient shapes
3. ✅ About — 4 pillars (automation, analytics, cloud, innovation), visual block
4. ✅ Services — 4 glassmorphism cards with hover glow effects
5. ✅ Features — 4 boxes with animated top-border reveal on hover
6. ✅ Statistics — Diagonal gradient band with 4 counters
7. ✅ Testimonials — 3 cards, center card highlighted/scaled
8. ✅ Pricing — 3 tiers, Pro plan featured with glow shadow
9. ✅ Contact — Form UI + contact info cards
10. ✅ Footer — Social icons, quick links, copyright

## Tech Used
- HTML5 (semantic tags)
- CSS3 only (no JavaScript, no frameworks)
- Google Fonts: Syne (display) + DM Sans (body)
- Flexbox + CSS Grid for layouts
- CSS Custom Properties (variables)
- CSS animations & transitions
- Glassmorphism via backdrop-filter
- Fully responsive (mobile / tablet / desktop)

## How to Run
Simply open `index.html` in any modern browser. No build tools needed.
