## Notes
- All images are optional — the site has elegant fallback visuals (animated orbits,
  gradient blocks, initials) if images are missing.
- Use `.webp` format for best performance; rename accordingly.
- For avatars, square images work best (they are displayed at 44×44 px in a circle).
- Free resources: Unsplash, Pexels, Icons8, Heroicons, Flaticon.
